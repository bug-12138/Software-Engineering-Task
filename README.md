# 🎓 CampusHub 校园互助服务平台

## 📖 项目简介 (Project Overview)
**CampusHub** 是一个专为高校学生打造的综合性互助服务与技能交换平台。

[cite_start]在日常校园生活中，学生群体存在大量诸如“快递代取”、“学习辅导”、“二手物品交换”、“活动组队”等高频互助需求。然而，目前这些需求多散落在微信群、QQ群及校园贴吧等碎片化渠道，普遍面临着**信息匹配效率低**和**缺乏安全信任机制**两大痛点 。

[cite_start]本项目致力于打造一个统一、高效、高信任度的闭环社区。在这里，学生可以自由切换“需求方”与“服务方”角色，发布任务、发现匹配、建立信任并完成协作。

## 🎯 核心功能 (Core Features)
[cite_start]为了在 10 周的敏捷开发周期内交付高质量的最小可行性产品（MVP），本系统当前重点聚焦以下核心模块：
* [cite_start]**🛡️ 用户与角色体系 (P0)**：支持学生认证注册与登录，允许用户在发布需求与接单服务两种角色间无缝切换。
* [cite_start]**📢 需求大厅 (P0)**：支持结构化发布互助需求（分类、地点、报酬等），并提供基于分类、时间和距离的列表检索与筛选功能。
* [cite_start]**🔄 订单状态机流转 (P0)**：实现从“接单确认” -> “进行中” -> “完成”的完整订单生命周期管理与双方确认机制。
* [cite_start]**⭐ 信用与评价体系 (P1)**：订单完成后强制双向评分，沉淀用户信用分，构建校园信任基石。

## 🛠️ 技术架构 (Tech Stack)
[cite_start]本项目采用前后端分离架构，注重系统的性能与可维护性。
* **前端 (Frontend)**：Vue3
* **后端 (Backend)**：Spring Boot (Java)
* **数据库 (Database)**：MySQL + Redis
* [cite_start]**基础设施与 CI/CD**：使用 Git 进行代码版本控制，并配置自动化构建与代码风格检查流水线。

## 🤖 AI 协同开发声明 (AI-Assisted Development)
[cite_start]本项目是基于「智协」全生命周期实训的探索实践。开发过程中深度整合了 GitHub Copilot 等 AI 编程助手与 AI 代码审查工具，以显著提升 CRUD 代码编写与测试用例生成的效率。团队严格遵守《AI 协作契约》，所有 AI 生成的代码均经过严格的人工 Code Review 后合入主分支。
